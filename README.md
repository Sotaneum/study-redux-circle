# Circle

## 요약

숫자를 더하거나 버튼을 클릭했을 때 색이 변하도록 합니다.

## 프로젝트 세팅하기

- 라이브러리 설치하기

```bash
npm install --registry https://registry.npmjs.com
```

## 프로젝트 실행하기

```bash
npm run start
```

## 프로젝트 빌드하기

```bash
npm run build
```

## 프로젝트 배포하기

```bash
npm run deploy
```
